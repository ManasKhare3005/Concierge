# Closing Day

Closing Day is an AI document concierge for real estate transactions, built for Lofty GlobeHack 2026.

This repository is being rebuilt cleanly in TypeScript with a `pnpm` workspace containing:

- `frontend` for the React/Vite client portal and agent dashboard
- `backend` for the Express/Prisma API, AI services, and SSE fanout
- `shared` for common domain types

Phase 1 currently covers scaffold and infrastructure. Product walkthrough docs land in Phase 8.
