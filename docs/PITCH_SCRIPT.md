# Pitch Script

## 3-Minute Demo Flow

### 0:00-0:25 - Open with the problem

"Agents do not lose time because documents exist. They lose time because confused clients ask scattered questions across email, text, and late-night panic. Concierge turns transaction confusion into a live, triaged workflow."

Show the agent triage board and point out:

- Sarah in `Needs Full Attention`
- Marcus and David in `Light Touch`
- Maria in `Clear`
- visible ROI ribbon at the top

### 0:25-0:55 - Show the client experience

Switch to Sarah's client portal.

"Sarah does not see raw PDFs and legal language only. She gets the actual transaction documents, a plain-English explanation, and a safe place to ask what the file means."

Open the purchase agreement and highlight:

- PDF preview
- plain-English summary sections
- AI live or fallback badge
- transparency / why expansion

### 0:55-1:20 - Trigger the real-time loop

Ask a worried question:

"I don't think I want this house anymore."

Then switch back to the agent view.

"The moment Sarah's tone changes, the dashboard changes too. The system answers the question, classifies it, generates a sentiment signal, recomputes readiness, and pushes the event to the agent in real time."

Point out:

- activity feed update
- Sarah's bucket and sentiment state
- routed-to-agent behavior

### 1:20-1:55 - Show agent control and trust

Open Sarah's detail or document workspace.

"This is not AI replacing the agent. The agent stays in control. Every explanation is transparent, editable, and reversible."

Edit a document summary and save it.

"If James wants to rephrase the explanation in his own voice, Sarah sees that update immediately in her portal."

### 1:55-2:35 - Show the voice bot

From Sarah's triage card, click `Call with bot`.

"When a real conversation is needed, Concierge can start a simulated outbound AI call flow that captures concerns, offers meeting slots, and prepares the agent before the actual human conversation."

Show:

- concerns and proposed slots in the modal
- bot session transcript
- audio or fallback-voice badge
- one click on a plausible client response

### 2:35-2:55 - Show the prep brief and repeat value

Confirm the booking and open the resulting brief.

"Now James does not walk into the call cold. He has the client's top concerns, tone, booking time, and a concise prep brief."

Jump to Repeat Clients.

"And the value does not stop at closing. Concierge keeps past clients visible as follow-up opportunities instead of letting that equity and referral value disappear."

### 2:55-3:00 - Close

"Concierge turns transaction confusion into agent leverage: fewer lost deals, fewer reactive calls, and more time spent where human judgment actually matters."
